# @shardix/transport

> Gateway WebSocket transport layer abstraction for **Shardix Framework**.

## Installation

```bash
pnpm add @shardix/transport @shardix/common
```

## Features

- Unified Gateway WebSocket event handler interface.
- Raw Discord interaction payload routing.
