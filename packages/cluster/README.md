# @shardix/cluster

> Cluster Manager and automatic Worker Shard calculation engine for **Shardix Framework**.

## Installation

```bash
pnpm add @shardix/cluster
```

## Features

- Automatic unique Shard allocation per worker (`shardId = index`, `totalShards = count`).
- Worker Threads and Child Process node abstractions.
- Healthcheck probes and crash auto-recovery.
