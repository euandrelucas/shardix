# create-shardix

> CLI project initializer for **Shardix Framework**.

## Quick Start

```bash
npx create-shardix my-bot
```

## Features

- Interactive project wizard for selecting Discord Library Adapter (Discord.js, Eris, Oceanic.js, Discordeno).
- Select Runtime (Gateway, HTTP Webhook, Hybrid, Distributed).
- Auto-configured pnpm monorepo or standalone TypeScript project.
