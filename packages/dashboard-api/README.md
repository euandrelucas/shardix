# @shardix/dashboard-api

> REST API Endpoints and token authentication server for **Shardix Dashboard**.

## Installation

```bash
pnpm add @shardix/dashboard-api
```

## Endpoints

- `GET /api/application`
- `GET /api/runtime`
- `GET /api/workers`
- `GET /api/shards`
- `GET /api/health`
- `GET /api/metrics`
- `GET /api/events`
- `GET /api/logs`
