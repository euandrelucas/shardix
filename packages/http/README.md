# @shardix/http

> High-performance HTTP Interactions Transport layer built on Fastify for **Shardix Framework**.

## Installation

```bash
pnpm add @shardix/http @shardix/common
```

## Features

- Discord Ed25519 signature verification hook.
- Fastify server integration for HTTP interactions webhook handling.
