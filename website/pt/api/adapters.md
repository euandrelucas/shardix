# Referência da API de Adapters

Adapters oficiais encapsulando bibliotecas de Discord no contrato padronizado `DiscordAdapter` do Shardix.

- `DiscordJSAdapter`: Encapsulado para Discord.js
- `ErisAdapter`: Encapsulado para Eris
- `OceanicAdapter`: Encapsulado para Oceanic.js
- `DiscordenoAdapter`: Encapsulado para Discordeno
