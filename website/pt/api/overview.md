# Visão Geral da API

Bem-vindo à referência de API do **Shardix Framework**.

## Pacotes Principais

- [`@shardix/core`](/pt/api/core): Container IoC, Fábrica da Aplicação, AutoScanner, Runtimes
- [`@shardix/common`](/pt/api/common): Decorators, Contratos, Interfaces
- [API de Adapters](/pt/api/adapters): `@shardix/discordjs`, `@shardix/eris`, `@shardix/oceanicjs`, `@shardix/discordeno`
- [API de Provedores](/pt/api/providers): `@shardix/provider-*`
