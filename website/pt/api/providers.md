# Referência da API de Provedores

Provedores modulares oficiais de infraestrutura para o Shardix Framework.

- `ConfigProvider`: Carregador de `.env` e configurações
- `LoggerProvider`: Logger estruturado Pino
- `CacheProvider`: Cache em Memória e Redis
- `EventBusProvider`: Mensageria Pub/Sub de eventos
- `HealthProvider`: Sondagens de saúde Liveness e Readiness
- `QueueProvider`: Filas de tarefas em segundo plano
- `ObservabilityProvider`: Métricas OpenTelemetry e Prometheus
- `DashboardProvider`: Servidor do Control Plane Dashboard
