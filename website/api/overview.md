# API Reference Overview

Welcome to the API reference for the **Shardix Framework**.

## Core Packages

- [`@shardix/core`](/api/core): IoC Container, Application Factory, AutoScanner, Runtimes
- [`@shardix/common`](/api/common): Decorators, Contracts, Interfaces
- [Adapters API](/api/adapters): `@shardix/discordjs`, `@shardix/eris`, `@shardix/oceanicjs`, `@shardix/discordeno`
- [Providers API](/api/providers): `@shardix/provider-*`
