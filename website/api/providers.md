# Providers API Reference

Official modular infrastructure providers for Shardix Framework.

- `ConfigProvider`: `.env` & config loader
- `LoggerProvider`: Pino structured logging
- `CacheProvider`: Memory & Redis caching
- `EventBusProvider`: Pub/Sub event messaging
- `HealthProvider`: Liveness & Readiness health probes
- `QueueProvider`: Background job queues
- `ObservabilityProvider`: OpenTelemetry & Prometheus metrics
- `DashboardProvider`: Control Plane Dashboard server
