# Introduction

**Shardix** is an enterprise-grade architecture framework for Node.js and TypeScript Discord bots and applications.

## Why Shardix?

1. **Zero Vendor Lock-in**: Your business logic is decoupled from Discord.js, Eris, Oceanic.js, or Discordeno.
2. **Layered Enterprise Architecture**: `Application -> Runtime -> Transport -> Adapter -> Providers`.
3. **Automatic Sharding & Distributed Workers**: Run seamless clusters across process threads and Docker containers without duplicate gateway events.
4. **Built-in Control Plane Dashboard**: Real-time monitoring, worker health, command analytics, and log streams.
